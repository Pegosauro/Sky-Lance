# Sky Lance — Vertical Shooter

Prototipo HTML originale di vertical shooter per iPhone, pronto per GitHub Pages.

## File inclusi

- `index.html` — gioco completo in un solo file HTML/CSS/JS.
- `manifest.webmanifest` — configurazione PWA.
- `sw.js` — cache offline base per GitHub Pages.
- `assets/icon.svg`, `assets/icon-192.png`, `assets/icon-512.png` — icone app.

## Pubblicazione su GitHub Pages

1. Crea una nuova repository su GitHub, ad esempio `sky-lance`.
2. Carica tutti i file di questa cartella nella root della repository.
3. Vai in **Settings → Pages**.
4. In **Build and deployment**, seleziona:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/root`
5. Salva.
6. Dopo la pubblicazione, apri l'indirizzo GitHub Pages da Safari su iPhone.
7. Tocca **Condividi → Aggiungi alla schermata Home**.

## Uso da iPhone

- Muovi la nave trascinando il dito sullo schermo.
- Lo sparo è automatico.
- Evita i colpi rossi.
- Raccogli i potenziamenti verdi e azzurri.

## Nota

Il gioco è originale e ispirato agli shooter arcade classici. Non usa asset, marchi o nomi protetti.
